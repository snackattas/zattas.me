# Automation Tool Logos

This directory contains logos for automation tools used in the Selenium Fun feature.

## Required Files

You need to add the following logo files (96x96px PNG with transparency recommended):

- `selenium-logo.png` - Selenium WebDriver logo
- `playwright-logo.png` - Playwright logo
- `cypress-logo.png` - Cypress logo
- `vibium-logo.png` - Vibium logo

## Where to Find Logos

- **Selenium**: https://www.selenium.dev/
- **Playwright**: https://playwright.dev/
- **Cypress**: https://www.cypress.io/
- **Vibium**: Check Vibium documentation or use Cypress logo as fallback

## Usage

These logos are displayed in the bouncing animation when automation is detected.
See: `src/components/SeleniumFun/BouncingLogo.tsx`
